This plugin changes the image in the top of login form.

== Description ==

Just create one image called wp-custom-login-logo.png and put in the plugin folder. After this Activate the plugin.
I've created this because i use wordpress as a CMS and sometimes users like their logo in the login form.
For more plugins check [my wordpress blog](https://antoniocampos.net)

== Installation ==

1. Upload the folder "wp-costum-login-logo" to the "/wp-content/plugins/" directory
2. Create your one wp-custom-login-logo.png and upload it to the plugin folder.
3. Activate the plugin through the 'Plugins' menu in WordPress
